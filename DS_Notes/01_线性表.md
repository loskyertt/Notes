# 1.重要知识点

1. 数据结构是相互之间存在 **一种或多种特定关系** 的数据元素集合。

2. 算法的五个特性：
   - 有穷性：算法必须在执行有穷步之后终止，即一个算法的操作步骤是有限的。
   - 确定性：算法中的每一条指令都必须都有确切的含义，并且对于特定的输入有特定的输出。
   - 有输入：算法有零个或多个输入，它们是算法开始前赋予参与运算的各个变量的初始值。
   - 有输出：算法有一个或多个输出，输出的值应该是算法计算得出的结果。
   - 可行性：算法是能够执行的，即算法中定义的操作都可以通过已经实现的基本运算执行有限次来实现。

3. 数据结构包括三个方面的内容：**逻辑结构**、**存储结构**、**数据的运算**。（容易把**数据的运算**忽略掉！！）

# 2.数组

顺序表：
- 插入：平均移动 $n/2$ 个元素，即 $\frac{n(n+1)/2}{n+1}$ 。
- 删除：平均移动 $(n-1)/2$ 个元素，即 $\frac{n(n-1)/2}{n}$ 。

推导：
插入有 $n+1$ 个位置，如：

```bash
  A   B   C   D     // 一共有 5 个位置
^   ^   ^   ^   ^
```

删除有 $n$ 个位置，如：

```bash
A B C D     // 一共有 4 个位置
^ ^ ^ ^
```

> 注意：**线性表**中元素的位序是从 `1` 开始的，而**数组**中元素的下标是从 `0`开始的。

# 3.链表

在单链表中设置头节点的作用是**便于插入和删除操作**。

**链表中头节点和头指针的关系**：不管带不带头节点，头指针始终指向链表的第一个节点。

示例（这里的`head`表示头指针）：

```bash
不带头节点的单链表：

head
 |
[1] → [2] → [3] → nullptr
```

```cpp
struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

ListNode* head = new ListNode(1); // 头指针指向第一个数据节点
head->next = new ListNode(2);
head->next->next = new ListNode(3);
```

```bash
带头节点的单链表：

head
 |
[X] → [1] → [2] → [3] → nullptr
```

> `X`代表头节点，不存储有效数据。

```cpp
struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

ListNode* head = new ListNode(INT_MAX); // 头节点（不存储有效数据）
head->next = new ListNode(1);
head->next->next = new ListNode(2);
head->next->next->next = new ListNode(3);
```

对于一个**带头节点**的循环单链表 `L`，判断该表为空表的条件是 `L->next = L`。
